2022年7月22日，Rita仍然是美少女

2022-07-23
docs shortcodes index

文章目录

2022年7月22日16:22，收到了Rita是美少女被全账号永久封禁的提示。

↑看来清华没有听取我的妙招。在这篇发表于3月14日的文章中，我向清华劝告，慌里慌张地知会平台删帖禁言多么窝囊，如果真想掩盖自己做的丑事，应该发挥身为最高学府的能量将Rita抓起来判个一千三百年有期徒刑。结果直到现在，他们还是毫无长进嘛。

早在今年三月，在清华大学连续发生多起丑闻而疯狂删帖时，就设想到了会有这样一天。这也是将这破公众号中与清华有关的内容集中收集备份到THUMachtFrei.top的原因。事实证明这一举动是极有远见的；这使得在Rita是美少女被永久封禁之后，其中最有价值的部分没有散失——这破公众号的五年间，发表了无数内容，学术杂谈、Rita的研究工作介绍、FAI轻小说、以及一切与Rita的日常生活相关的东西；然而这些都不是最重要的。Rita的学术研究不需要被人所知，一百年后自然会有人从故纸堆里翻出来评判；Rita的生活更是无足轻重。唯独那些与清华有关的部分是最珍贵的、最不可散失的，因为它被无数人所共享，与无数人的生活息息相关，记录着清华这所早已变质异化腐臭难闻的建筑所作的每一处恶行。只要它们仍然被记录着，清华的校领导就将始终背负着他们理应背负的十字架，终日不得安寝；清华上中下层领导的恶臭、就将散发出去，从而加速这所学校内表面上是昏聩荒唐，实质上是漠视生命、物化人格的风气被涤荡一新的那一天的到来。到这一天为止，清华园内的悲剧都将不断地重复上演，而我早就发誓要不惜一切代价保护这里的学生，因为我是如此地深爱他们，我存活至今便是为了他们而活，我生命中的一切都将为避免悲剧的再上演而献上。

所以，我仍然在这里。用一个注册第一天的公众号，从零开始。请不要觉得我们的力量减弱了；一些人的力量是否强大，从来不取决于他们人数多寡，有多少关注者、多少追随者，也不取决于他们是否身居高位，把握着他人的命运。它取决于——它仅仅取决于他们是是否是站在正确的一方。他们所面对的对手越是凶恶，越是下流无耻，越是荒诞而暴戾，他们就越是强大，因为这愈发彰显出他们的正确性。请不要畏惧，因为历史终究会做出选择，历史终究会站在正确的一方。

“世上只有一种道理，即加害者将付出代价，受害者将得到补偿，除此之外，绝无其他。”——《在关于抵制清华大学之行动呼吁的半年之后》2022年3月11日

“如果自觉是清白的话，如果自觉无愧的话，如果自觉未曾作恶的话，如果自觉这些不是应得的话，就来抓捕我罢，我会在这里带着微笑等待着。” ——《关于三月事件的再回忆》2021.12.04

“所以，请大家起来罢，这是我恒久的恳求，我的双膝早已跪在地上，面向着清华的受害者，呼喊着站起身来，向清华一推就倒的建筑施以重拳。”——《在关于抵制清华大学之行动呼吁的半年之后》2022年3月11日

“你校打不垮我，你校能弄垮的只有它自己。” ——《2021年3月21日 清华大学把我送进了拘留所》2021.03.28
