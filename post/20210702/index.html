链接：自发与形式——在“拉歌”之后

2021-07-02
docs shortcodes index

文章目录

导读： 此文是对激光笔事件后某同学所作文章的评论。

原想转载，原作者似乎忘记申请原创，只能以链接的形式分享：https://mp.weixin.qq.com/s/-xrJUnu6LIYq_qmX_aaW3Q

零字班同学是英雄的，堪大用的。在经历了激光笔照眼这般恶劣的对待后，仍然坚持完成军训，履行法定义务，此为堪大用之处；与此同时不停息地进行了抗争和辩论，此为英雄之处。有人认为，在军训期间就要依照军营的规定，不应用日常生活的道德规范来指责军训领导的行为；持这种论调的人不妨想一想，一支人民军队中是否会有在和平时期因队列不整（且很大程度上是由于长官没有做好准备）而用武器对准自家战士的行为，即使这武器不是真枪实弹，而是足以致盲的大功率激光笔；想一想这种事情是否会出现在一支人民军队的非战斗减员理由中。就此事，同学们持续进行着英勇的抗争，即使按照你校一贯的做法终究不会给出答复，即使这些抗争在几个月之后就会被造成这些事故的校领导抛之脑后，其述职报告上只会剩下“成功组织零字班军训”的荣耀履历，即使这些事故在不久后就会被彻底抹消，学校继续操弄世人的记忆，仿佛一切都没有发生过一般招揽毫不知情的新生。即使如此，同学们依然在团结地抗争着。

等到多年之后，作为这次的和一直以来所有的事件的元凶的、被异化为行政机构不可或缺又随时可被替代的零件的可怜的校领导接受更高的荣誉之时，他们或许不会记得这些抗争。可是历次事件的痕迹总不会被彻底遗忘，亲历者们会带着真实的故事活下去，带着对清华究竟是怎样的面目的深刻的记忆活下去，这些事件一再发生，这份记忆也就一再深化，直到它们终究被大众所共享。这一天便是维系当今清华的建筑的崩塌之日。

文章作者 Rita是美少女
